★case 함수 :
형식 :
case 열 또는 데이터
when 조건1 then 조건1가 참일때 반환결과
when 조건2 then 조건2가 참일때 반환결과
when 조건3 then 조건3가 참일때 반환결과
 else 위 조건에 해당되지 않을때 반환할 결과
end as 별칭

문제22) (case 함수 사용) jobs 테이블의 job_title 중 'President'의
max_salary를 2.5배 인상, 'Programmer'의 max_salary를 1.5배 인상,
나머지는 0.02인상하는 테이블을 UPSALE열로 출력하시오.

SQL> select job_title, max_salary, case job_title when 'President' then max_salary *2.5
  2  when 'Programmer' then max_salary*1.5
  3  else max_salary*0.98 end as upsale from jobs;

★ sum 함수 :

문제23) job테이블의 max_salary의 값을 모두 합하시오
SQL> select sum(max_salary) from jobs;

문제24) jobs테이블의 job_title이 '~manager'로 끝나는 직업의 max_salary 값을 모두 합하시오
SQL> select sum(max_salary) from jobs
  2  where job_title like '%Manager';

★ count 함수 : 데이터 개수를 구해주는 함수

문제25) jobs테이블의 개수를 출력하시오
select count(*) from jobs;

문제26) jobs테이블의 max_salary값이 10000이상인 데이터의 개수를 출력하시오
SQL> select count(max_salary) from jobs
  2  where max_salary >= 10000;

★ max 함수 : 최대값을 구하는 함수
★ min 함수 : 최소값을 구하는 함수

문제27) jobs테이블의 max_salary값 중 가장 큰 값을 출력하시오
SQL> select max(max_salary) from jobs;

문제28) jobs테이블의 min_salary값 중 가장 작은 값을 출력하시오
SQL> select min(min_salary) from jobs;

★ avg 함수 : 평균을 구하는 함수

문제29) jobs테이블의 max_salary의 평균값을 출력하시오.(소수 둘째자리까지 출력)
SQL> select avg(max_salary) from jobs;
SQL> select round(avg(max_salary),2) from jobs;

문제30) jobs테이블의 job_title이 '~manager'로 끝나는 직업의 max_salary 평균값을 출력하시오
SQL> select avg(max_salary) from jobs
  2  where job_title like '%Manager';
