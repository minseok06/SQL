★ 조인한 테이블에서 조건을 주어서 출력해 보기
select 고객.고객번호, 고객.이름, 주문.주문번호, 주문.주문가격
from 고객, 주문
where 고객.고객번호 = 주문.고객번호;

★ 위 조인 복사해서 new라는 테이블 복사생성하여 출력
create table NEW as select 고객.고객번호, 고객.이름, 주문.주문번호, 주문.주문가격
from 고객, 주문
where 고객.고객번호 = 주문.고객번호;

문제1> 'NEW' 테이블에서 주문가격이 5000 이상인 자료의 '고객번호'와 '이름'을 출력하시오.
SQL> select 고객번호, 이름 from new
  2  where 주문가격 >= 5000;

문제2> 'NEW' 테이블에서 이름이 '김성팔 인 자료를 출력하시오.
SQL> select * from new
  2  where 이름 = '김성팔';

문제3> '고객' 테이블에서 '김성팔'의 이름과 비밀번호 열만 출력하시오.
SQL> select 이름, 비밀번호 from 고객
  2  where 이름 = '김성팔';
