create table test1 as select * from jobs;

☆update문 : 테이블에 있는 데이터 수정하기
update 테이블명
set 변경할 열1=데이터, 변경할 열2=데이터, ..... 변경할 열n=데이터
where 데이터를 변경할 대상 행을 선별하기 위한 조건

문제31> min_salary값 전체를 9870으로 수정하시오
SQL> update test1
  2  set min_salary = 9870;

문제32> 새로운 테이블에서 job_title이 Manager로 끝나는
max_salary값을 7770으로 수정하시오
update test1
set max_salary = 7770
where job_title like '%Manager';

☆delete문 : 테이블에 있는 데이터 삭제하기
SQL> delete from test1
  2  where job_title like '%Manager';

★데이터 정의어(DDL) : 객체를 새로 만들거나 기존에 존재하던 객체를
생성, 변경, 삭제하는 등의 기능을 수행하는 명령어를 데이터 정의어라 한다.
CREATE : 테이블 생성
ALTER : 이미 생성된 객체를 변경할때 사용
DROP : 객체를 삭제하는 명령

☆ CREATE TABLE 소유계정, 테이블이름(
열1 이름 자료형,
열2 이름 자료형,
열3 이름 자료형,
....
열N 이름 자료형);

※ 정수 타입 NUMBER(3)
※ 문자 VARCHAR2(5)

※ 테이블 이름 생성 규칙
1. 테이블 이름은 문자로 시작해야 한다. (한글도 가능하며 숫자로 시작할 수 없음)
2. 테이블 이름은 30byte 이하여야 한다. (즉 영어30자, 한글은 15자까지 사용가능)
3. 같은 사용자 소유의 테이블 이름은 중복될 수 없다.
4. 테이블 이름은 영문자(한글가능), 숫자(0~9)와 특수문자를 사용할 수 있다.
5. SQL 키워드는 테이블 이름으로 사용할 수 없다.(함수 이름 SELECT, FROM등은 테이블
이름으로 사용 불가)

※ 열 이름 생성 규칙
1. 열 이름은 문자로 시작해야 한다.
2. 열 이름은 30byte 이하여야 한다.
3. 한 테이블의 열 이름은 중복될 수 없다.
4. 열 이름은 영문자 (한글가능), 숫자(0~9)와 특수문자를 사용할 수 있다.
5. SQL 키워드는 열 이름으로 사용할 수 없다.(함수 이름 SELECT, FROM등은 열 이름으로
사용 불가)

문제34> '성적표' 테이블을 생성하고
성명, 국어, 영어, 수학, 그래픽디자인, DB, 프로그래밍, 평가 의 열을 생성하시오.
(단 성명과 평가는 문자형 타입선언, 나머지는 숫자형 3자리로 선언하시오)
※ 정수 타입 NUMBER(3)
   문자 타입 VARCHAR2(5)

create table 성적표(
성명 varchar2(8),
국어 number(3),
영어 number(3),
수학 number(3),
그래픽디자인 number(3),
DB number(3),
프로그래밍 number(3),
평가 varchar2(5));

문제34> '성적표' 테이블에 10개 이상의 데이터를 insert문을 이용해
채워넣으시오. 평가는 공백
insert into 성적표(성명, 국어, 영어, 수학, 그래픽디자인, DB, 프로그래밍, 평가)
